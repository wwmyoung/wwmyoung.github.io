    CherryStudio中配置MCP-word服务

    MCP CherryStudio MCP Word
    发布时间 : 2025-04-20 00:00
    阅读 :
    评论:
    1. 说明
    2. 配置

    说明

    最近关于MCP热议不断，有人将其比喻为AI大模型领域的TypeC接口。为了跟上技术，遂研究在CherryStudio中如何使用MCP服务工具。这里就从日常使用最多的office套件开始。

    关于MCP的相关技术说明请参考官方文档，并且默认已安装CherryStudio。

    配置

    这里用到的MCP服务工具名称是mcp-server-office，有关其介绍可点击CherryStudio客户端中的设置->MCP服务器，然后点击右上角的更多MCP，进入社区后直接搜索关键词即可看到。

    点击添加服务器后，按照该工具说明文档配置即可。其中“名称”“描述”可以随意填写，剩下的选项要按配置文档填写，每个选项具体含义参照CherryStudio官方说明。

    配置完成后，点击保存，待服务显示变绿，即可使用。

    回到CherryStudio的对话界面，开始对话前选择MCP服务器，勾选刚才添加的office-word服务器。

    在对话框中输入需求，即可完成对应操作。需要说明的是，对于word的创建以及内容修改，需要在对话时指定具体路径。

    服务如果配置成功，并且大模型可用的情况下，会出现上图中已完成的提示。若失败，可点开查看具体失败的原因进行修改，或者在设置中，找到数据设置，查看应用日志的报错原因。成功之后，点开相应位置，即可找到对应文档。

    插入文档中文字的字体效果等，需要自己对话时提前给出。至于修改文档内容，操作步骤一样。在对话框中指定具体路径下的文件，键入相应的需求即可。
