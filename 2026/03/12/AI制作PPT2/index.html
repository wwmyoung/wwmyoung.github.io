    AI制作PPT--基于html语言--案例分享

    PPT AI PPT
    发布时间 : 2026-03-12 10:55
    阅读 :
    评论:
    1. 工具
    2. 效果

    继上次分享过使用html语言制作PPT演示文稿后，又看到许多许多博主分享这方面的经验。此次案例分享加入了一些新的风格提示词。

    工具

    IDE：VS code + claude code

    大模型：minimax的coding plan，模型是2.5

    新加效果的提示词：

    1. 毛玻璃卡片（glass-card）：background: rgba(255,255,255,0.7) + backdrop-filter: blur(12px) + 半透明边框
    2. 光晕背景（glow-bg）：绝对定位的大圆形色块，filter: blur(100px)，半透明，营造氛围感
    3. 网格底纹：用 linear-gradient 画 40px 间距的浅色网格线作为全局背景
    4. 颜色渐变横幅：页面底部的结论性文字用渐变色或强调色的圆角大卡片承载

    大纲内容：通过kimi大模型，调研的2025年6月到2026年3月AI发展趋势

    效果

    截图中可能不太清楚，每页PPT加入了网格背景，表格页显示也基本完整。整体效果基本可用，没有需要修改的排版问题。
